# Python SAST Dashboard

React-based dashboard for visualizing SARIF findings generated by the Python SAST Scanner.

## Live Demo

https://scan-python-safe.base44.app

## Run Locally

```bash
npm install
npm run dev
```

## Technologies

* React
* JavaScript
* Vite
* Base44

## Purpose

The dashboard allows users to:

* Upload SARIF reports
* Explore security findings
* View vulnerable code snippets
* Visualize taint-flow paths
* Inspect vulnerability details
